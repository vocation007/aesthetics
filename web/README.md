# Web

This folder stores the web-first outputs for “审美的逻辑”.

- `web-html/index.html`: webpage directory and issue index.
- `web-html/daily-lessons/`: standalone webpage article HTML.
- `web-html/assets/`: assets used by webpage articles.
- `WEB_MEMORY.md`: persistent context for domain, publishing state, growth goals, and web decisions.

The restored daily workflow generates one new web article here and updates the index.
